#  Global super store Sales Analysis

## Project Overview

This project analyzes a global super store sales dataset using Power BI.

The dataset contains sales transactions from store branches across multiple countries.

The objective is to create a detailed and interactive sales report for stakeholders.

## Dataset

Source: Google Sheets

Link: https://docs.google.com/spreadsheets/d/1Ez8RhO1rE2QikfdddOZ19eEZhN4r6eW7WHb9qB05T7E/edit?usp=sharing

## Data Cleaning

Basic data cleaning was performed using Power Query in Power BI.

Missing values were handled appropriately.

Duplicate records were removed.

Column names and data types were standardized.

## Data Segmentation

Sales visuals were segmented using:

Country

Region

Market

Slicers were added to enable interactive filtering.

## Shipping Mode Analysis

Analyzed shipping patterns by visualizing the percentage of orders based on ship mode.

Pie/Donut charts were used to show shipping distribution.

Identified the most frequently used shipping methods.

## Sales Performance Report

Sales were analyzed across:
City

State

Region

Market

Bar and column charts were used for comparison.

Drill-down functionality was enabled for deeper analysis.

## Tables for Visualizations

Tables were created for all visualizations.

Tables provide exact values to support charts and insights.

## Interactive Sales Dashboard

All visuals were combined into an interactive Power BI dashboard.

Slicers and filters allow stakeholders to explore data easily.
